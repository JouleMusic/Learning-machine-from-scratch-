# 数据清洗

## 基本概念
一个谚语：
> garbage in, garbage out

所以好的训练集输入直接影响训练出的模型的泛化能力。于是在实际中，数据清洗这个过程会花掉一大部分时间，而且它会使得你对于业务的理解非常透彻。

简而言之：
> 数据清洗做的事情 => 去掉脏数据

> (2)一个月买脸盆墩布买了10w的人
